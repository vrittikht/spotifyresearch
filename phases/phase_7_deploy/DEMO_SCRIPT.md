# 5-Minute Demo Script

Use this walkthrough for portfolio presentations and case study videos.

| Step | Time | What to show |
|------|------|--------------|
| 1 | 30s | Open live URL. Problem: discovery feedback is scattered across Reddit and app stores. |
| 2 | 30s | **Overview** — 300+ reviews, sentiment split, executive summary, source breakdown. |
| 3 | 1min | **Evidence** — filter by keyword (e.g. "discover weekly"), expand a row, show structured AI extraction. |
| 4 | 1.5min | **Themes** — top themes with user quotes; mention algorithm repetition, personalization gaps. |
| 5 | 1.5min | **Research Report** — walk all 6 PM research questions with evidence counts. |
| 6 | 30s | **Segments** — compare `casual_listener` vs `algorithm_skeptic` barriers. |
| 7 | 30s | Close: *"This research informs Part 2 — problem definition and solution design."* |

## Talking points

- **Not an MVP** — a research instrument for PM discovery work
- **Real public data** — Reddit RSS + Play Store scrape, no Reddit API
- **Groq extraction** — structured JSON per review (barriers, segments, sentiment)
- **Synthesis** — themes + six research answers stored in Supabase
- **Reproducible** — full pipeline via Admin or CLI

## Optional: show Admin (30s)

Briefly show pipeline status and that collection → analysis → insights is runnable end-to-end.
